JavaScript

1,JS与HTML，CSS有什么区别？
    HTML就是一种标签语言，HTML就是给网页去提供内容。是运行在浏览器端。
    CSS层叠样式表，是网页提供布局和样式。是运行在浏览器端。
    JS也可以运行在浏览器上。浏览器是JS一个运行环境。
    HTML，CSS不能叫编程语言，JS一门真正编程语言。 程序员--->精通一门编程语言。

2,编程语言的分类：
    后端：JAVA，Python，C++，PHP
    前端：JavaScript，

    未来10年死不了语言：JavaScript，C++，JAVA，Python（web后端，数据分析为主）（AI(算法（算法应用，算法研究）)）

3，JS现在的运行环境：
    在很早之前，JS只能运行在浏览器端，主要写去网页的动态效果，说白了，就是去
    做一些网站的特效。但是JS现在可以运行在如下的环境：
    小程序，公众号，网站
    JS还可以做后端。JS要运行在后端，它是运行在Node上面的，也就是说Node和浏览器一样，是一个运行环境，只不过，浏览器是前端JS的运行环境，node是后端的JS的运行环境。
    JS可以做原生app  React Native    Weex   
    JS还可以做PC端的软件
    JS目前可以称为一门全栈式的编程语言：
        web前端，webapp
        小程序，公众号 
        原生app
        写后端
        PC端APP

4，写JS代码，以web前端，和webapp为主，那么我们的JS代码也是运行在浏览器端。
